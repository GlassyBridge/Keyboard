<h1 align="center"> Wireless-Modular Keyboard </h1>
<div align="center"><img width="80%" alt="3D view" src="https://github.com/user-attachments/assets/8fd15b56-dfcd-42e9-b987-81b10737f17e"/></div>
<p align="center">This project is a custom modular keyboard that has both wireless and wired capabilities. It also uses low profile, kailh choc hotswap switches increasing customizability and its overall asthetic. Additionally, it offers magnetic connection following the I2C protocol which ensures any type of module can connect to the main board effectively and securely.</p>
<h2></h2>
<h2> Features </h2>
<ul>
  <li>Works over USB-C (wired) or Bluetooth (wireless)</li>
  <li>Integrated magnetic pogo pins for stable I2C connection and adding external modules (numberpads, macropads or OLED modules)</li>
  <li>Integrated charging circuit for portability and wireless operation</li>
  <li>Uses Kailh Low Profile Choc hotswap sockets for key switch connections</li>
  <li>Features SK6805 addressable LEDs for customizable per key RGB lighting</li>
  <li>Includes a dedicated SWD interface for debugging and development</li>
  <li>A dedicated slide switch to manage battery life effectively</li>
</ul>
